## 1. 事業コンセプト

### コンセプト一言（確定）
> **「ホームページ、見てから決めませんか。」**

サブコピー（確定）：
> ITが苦手でも大丈夫。沼田の、相談できるホームページ屋さんです。

### 核となる価値 ＝「安心」
1. **失敗しない安心** … お金を払う前に、完成イメージ（無料デモ）が見られる
2. **わかる言葉で進む安心** … 専門用語ゼロ、ITが苦手でも置いていかれない
3. **地元の人に頼める安心** … 顔が見える、相談しやすい、同じ沼田の人

> 一言でいうと『**安心して持てる、はじめてのホームページ**』。
> 集客・売上アップは、その安心の先にある“結果”として伝える（安心が主・集客が従）。

## 3. 他社との違い（差別化）

### 基本方針
**価格では戦わない。「安心して頼める人かどうか」で差別化する。**

### 比較表
| | 一般的な制作会社 | ネットの格安サービス | このサービス |
|---|---|---|---|
| 料金 | 数十万円〜 | 安いが内容は不安 | 低価格（初期+月額） |
| 完成イメージ | 契約後に初めて見る | テンプレ任せ | ✅ 払う前に無料デモで見られる |
| やりとり | 難しい打合せ | ほぼメールだけ | ✅ 地元・顔が見える・相談しやすい |
| IT知識 | ある前提で話される | 全部自分で | ✅ 苦手でもOK、言葉をかみ砕く |
| 対象 | 中〜大規模も | 誰でも（焦点なし） | ✅ 小規模・地域のお店に専念 |
| 納期 | 数週間〜数ヶ月 | まちまち | ✅ AI活用で短納期 |

### 5つの違い（お客さんの言葉で）
1. 見てから決められる → 気に入らなければ断ってOK。ムダにならない
2. アンケートに答えるだけ → 難しい打合せナシ。忙しくても大丈夫
3. 地元・沼田で相談できる → 困ったらすぐ聞ける。顔の見える相手
4. ITが苦手でも置いていかない → 専門用語は使わない。作業は全部こちら
5. 小さなお店だけに集中 → 大企業向けじゃない。あなたのお店にちょうどいい

### AIの見せ方（重要ルール）
AIは「低価格・短納期の理由（裏方）」として控えめに見せる。前面で押すとITが苦手な層には冷たく映る。
- ❌「最新AIがあなたのHPを自動生成」
- ⭕️「AIをかしこく使うから、この値段・この早さ。仕上げはちゃんと人がやります」
- → **「AIで効率化 × 人が寄り添う」** のバランス。

## 4. LP構成（セクション順）

訪問者の心理：「自分に関係ある」→「気になる」→「無料なら試したい」→「申し込もう」

| # | セクション | 訪問者の心の声 | 役割 |
|---|---|---|---|
| 1 | ファーストビュー（メインコピー+デモ申込ボタン） | 「お?見てから決められるの?」 | 3秒でつかむ |
| 2 | 共感・課題提起 | 「まさに私のことだ…」 | 自分ごと化 |
| 3 | サービス紹介 | 「なるほど、そういうことか」 | 正体を明かす |
| 4 | 無料デモHPの流れ（3ステップ） | 「思ったより簡単そう」 | ハードルを下げる |
| 5 | 選ばれる理由（5つの違い） | 「他と違うんだな」 | 信頼を積む |
| 6 | 制作イメージ・実績（サンプルデモ） | 「こんな感じになるんだ」 | 不安を消す |
| 7 | 料金（3プラン） | 「これなら払えそう」 | お金の不安を解消 |
| 8 | よくある質問 | 「なら、大丈夫かな」 | 背中を押す |
| 9 | 代表プロフィール | 「この人になら頼めそう」 | 人で信頼させる |
| 10 | 申し込みCTA | 「よし、試してみよう」 | 行動させる |

### 設計意図
- ①の直後に②共感（いきなり売り込まない）
- ④デモの流れを早めに置き「自分にもできそう」と思わせてから理由（⑤）を語る
- ⑥サンプルデモは料金の“前”に置き、価格を「お得」に感じさせる
- ⑨プロフィールは終盤で「誰がやるの?」の不安を解消
- **CTAは最低3回**（①直後・⑦料金後・⑩最後）。スマホは申込ボタンを下部に常時表示
- 全体トーン：終始「断ってもいい」空気。ボタンは「まずは無料でデモを見てみる」系の軽い言葉

## 6. 各セクションの文章方針（確定文案）

文章ルール：結論ファースト・やわらかい語りかけ・「断ってOK」を繰り返す・専門用語ゼロ。

### ① ファーストビュー
狙い：3秒で「自分に関係ある+怖くない」。
```
# ホームページ、見てから決めませんか。
ITが苦手でも大丈夫。沼田の、相談できるホームページ屋さんです。
アンケートに答えるだけで、完成イメージを無料でお作りします。
[ ▶ まずは無料でデモを見てみる ]
```

### ② 共感・課題提起
狙い：売り込む前に悩みの代弁で心を開く。自分を否定されないこと。
```
## “そろそろウチもホームページ…”で、止まっていませんか?
- 作りたいけど、高そうで踏み出せない
- パソコンは苦手。自分には無理かも
- 忙しくて、そんな時間はとれない
- 何を載せればいいのか、よく分からない
——大丈夫です。その「止まっていた理由」を、ぜんぶ無くしました。
```

### ③ サービス紹介
```
## アンケートに答えるだけ。あなたのお店のホームページが、形になります。
難しい打ち合わせも、専門用語も必要ありません。
いくつかの簡単な質問に答えていただくだけ。あとはこちらで、お店の“顔”になるホームページを作ってお見せします。
しかも——できあがりを見てから、頼むかどうか決められます。
```

### ④ 無料デモHPの流れ
```
## 申し込みから完成イメージまで、たったの3ステップ。
① 申し込む … フォームに必要事項を入力（1分）
② アンケートに答える … お店のことを簡単な質問でお聞きします（10分ほど）
③ デモを見る … 数日後、完成イメージのホームページをお届け
気に入ったら、そこから正式にご契約。気に入らなければ、断っていただいてOKです。
```

### ⑤ 選ばれる理由
```
## “安いから”ではなく、“安心だから”選ばれます。
1. 見てから決められる — お金を払う前に完成イメージが見られるから、失敗がありません。
2. 答えるだけでOK — 難しい打ち合わせはナシ。忙しくても大丈夫。
3. 地元・沼田で相談できる — 顔の見える相手だから、困ったらすぐ聞けます。
4. ITが苦手でも置いていきません — 専門用語は使いません。むずかしい作業は全部こちらで。
5. 小さなお店のために — 大企業向けではなく、あなたのお店にちょうどいいホームページを。
```

### ⑥ 制作イメージ・実績
狙い：百聞は一見。サンプルデモを“証拠”として見せる。
※現時点は実顧客がいないため「お客様の声」は出さず、サンプル/自作デモを“作品（見本）”として堂々と見せる。
```
## たとえば、こんなホームページができます。
言葉だけでは伝わりにくいので、実際の制作イメージをご覧ください。
（飲食店・美容室など、業種別のサンプルデモを掲載）
「自分のお店だったら、どうなるかな?」——それを無料で見られるのが、このサービスです。
```

### ⑦ 料金（→ 詳細は「8. 料金・コース設計」参照）
狙い：「これなら払える」と安心させる。シンプルに、隠さず、安心を前面に。
```
## はじめやすい価格に、しました。
### 月額5,500円〜。作って終わりではなく、公開後もずっとそばで。
ホームページは「作って終わり」ではありません。月額にはこんな安心が含まれます。
- ✅ 見てから決められる … デモを見て、納得してからご契約
- ✅ 公開後も相談できる … 「ここ直したい」にいつでも対応
- ✅ ちょっとした修正も頼める … 営業時間やメニュー変更もお任せ
- ✅ 困った時に相談できる … 分からないことは、何でも聞いてください
- ✅ 地元で頼れる … 同じ沼田だから、すぐ近くに相談相手がいます
（3プランを高い順に表示）迷ったら「スタンダード」がおすすめです。小さなお店に必要なものが、ちょうどぜんぶ揃っています。
```

### ⑧ よくある質問
狙い：申し込み直前の“最後の小さな不安”を先回りで消す。
```
## 不安なこと、先にぜんぶお答えします。
Q. 本当に無料でデモを見られますか? — はい。デモまでは一切費用はかかりません。
Q. パソコンが苦手でも大丈夫? — 大丈夫です。スマホで質問に答えるだけでOKです。
Q. 気に入らなかったら? — 遠慮なく断ってください。費用は発生しません。
Q. 沼田市以外でも頼める? — 周辺地域も対応しています。お気軽にご相談ください。
```

### ⑨ 代表プロフィール
狙い：個人運営の不安を「人柄・地元」で安心に変える。※顔写真は使わず**似顔絵イラスト**を使用。塾の経歴はここで軽く触れる程度（前面には出さない）。
```
## 沼田で、あなたのお店の“ホームページ係”になります。
はじめまして。沼田市出身、合同会社4Uの代表です。
普段は地元で個別指導塾を運営しています。「難しいことを、分かりやすく伝える」のが仕事です。
その経験を活かして、ITが苦手な方にも、つまずかないようご案内します。
同じ地域で商売をする者として、あなたのお店の力になれたら嬉しいです。
```

### ⑩ 申し込みCTA
狙い：プレッシャーを与えず、軽く背中を押す。
```
## まずは“タダで見てみる”から、始めませんか。
しつこい営業は一切しません。気に入らなければ、それで大丈夫。
だからどうぞ、気軽にお試しください。
[ ▶ 無料でデモHPを申し込む ]
```

## 7. デザインの方向性

> 詳細なカラー／タイポは `lp/brand-guidelines.md` に準拠。ここでは方針を要約。

全体結論：**「きれいでオシャレ」より「やさしくて分かりやすい」**。“近所の親切な人”の雰囲気。

- **色使い**：ベースは白／生成り。メインはやさしい青（Trust Blue）。差し色のアンバーは**ボタンだけ**。赤多用・黒背景・ギラギラは避ける。
- **写真・イラスト**：やわらかい手描き風イラスト中心。写真は沼田の風景・笑顔の人物。無機質IT素材・AI丸出し画像は避ける。
- **レイアウト**：縦1本道。1メッセージ=1ブロック。重要箇所はカード型。
- **文字サイズ**：大きめ（本文16〜18px以上）。年配の店主も読める。短文+改行でリズム。
- **余白**：たっぷり（=ゆとり・親切の表現）。
- **スマホ**：最優先。ボタンは指で押しやすく画面幅いっぱい。申込ボタンは下部に常時表示。
- **信頼感の出し方**：代表の似顔絵イラスト／「断ってOK」「無料」を繰り返し明記／サンプルデモを実際に見せる。
- **地域密着感**：「沼田」の地名を各所に／沼田の風景・ローカルな言葉／「同じ地域で商売する者として」のご近所目線。

## 8. 料金・コース設計（確定）

### 方針
- ゴルディロックス効果（松竹梅）で**真ん中の「スタンダード」を本命**に。
- 表示順は**高い順**（プレミアム → スタンダード → ライト）。
- LP上の入口価格表現はすべて **「月額5,500円〜」**。
- LP/HPという言葉は使わず、**たとえ（1枚／複数ページ）で見せる**。
- コピーは安さだけでなく「見てから決められる・公開後も相談・修正も頼める・困った時に相談・地元で頼れる」の安心を前面に。

### プラン（高い順）
| | プレミアム | ⭐スタンダード（一番人気・本命） | ライト |
|---|---|---|---|
| 中身 | 複数ページの本格サイト（3〜5ページ目安） | 充実した1ページ | 簡易な1ページ |
| たとえ | メニューや実績などページを分けてじっくり見せる | 1枚にお店の魅力たっぷり。これだけで“ちゃんとした顔” | お店のことを1枚にコンパクトにまとめる |
| 初期費用（税込） | **0円** | **0円** | **0円** |
| 月額（税込） | 応相談 | **8,800円** | **5,500円** |
| 位置づけ | 手厚いが少し大きめ（デコイ） | 必要十分で安心（本命） | 最低限・お試し |

- プレミアムは固定価格を出さず**「応相談」**（受注後に作り込む。デコイ役を文章で果たす）。

### スタンダードに誘導する仕掛け（採用：1〜4）
1. **ライトに“物足りなさ”を残す**：簡易1ページ／更新は軽微のみ／写真・原稿サポートは最低限、と正直に書く。
2. **ライトに注釈**：「※まずはお試しで、という方に。物足りなければ後でスタンダードに変更もできます」。
3. **スタンダードに“数字の安心”**：「迷ったらこれ」＋（実績が出たら）「お申し込みの約8割がスタンダード」。
4. **差額を“1日あたり”に割って見せる**：ライト→スタンダードの月額差3,300円を「1日あたり約110円」で心理的に縮める。
（5「ライトの枠制限」は不採用）

### 視覚演出
- スタンダードの列のみ色付き・少し大きく・「一番人気」バッジで主役化。
